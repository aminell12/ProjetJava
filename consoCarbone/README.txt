                                                            README
Voici une partie de notre projet portant sur l'empreinte carbone d'un individu. Cette dernière est basée sur plusieurs critères notamment l'alimentation,
la consommation des biens, le transport...

                                                            Compilation/Exécution
Afin de compiler notre programme il suffit de se situer dans le terminal, en dehors du dossier contenant nos fichiers, donc dans le fichier parent de consoCarbone.
Pour le compiler il faut entrer la commande suivante : javac consoCarbone/*.java afin de compiler tous les fichiers de type java dans le dossier.
Pour exécuter le programme il faut entrer la commande suivante : java consoCarbone/main afin d'exécuter le main de notre programme.

Afin de vérifier cette partie du projet, nous avons effectué de simples tests qui permettent de vérifier la fiabilité nos méthodes, nos calucls...
Nous avons décidé d'interagir avec l'utilisateur plus tard dans le projet.
